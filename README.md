---

## 🚀 Live Demo

🌐 **Streamlit App:**

---

## 📌 Project Overview

Global conflicts influence much more than military operations—they affect national economies, inflation, energy markets, and geopolitical stability.

This project integrates multiple real-world datasets to analyze how armed conflicts impact the global economy through interactive dashboards and advanced data visualizations.

The application allows users to explore historical conflict patterns, military expenditure, inflation trends, oil prices, and country-wise impact scores.

# ▶️ Installation

### Clone Repository

```bash
git clone https://github.com/Prasanna99-rgb/global-conflict-economic-impact-dashboard.git
```

### Navigate to Project

```bash
cd global-conflict-economic-impact-dashboard
```

### Install Requirements

```bash
pip install -r requirements.txt
```

### Run Application

```bash
streamlit run app.py
```
